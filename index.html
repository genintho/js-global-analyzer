Js-global-analyzer

An easy to use and very simple utility lib to help you understand what is going on in the global Namespace of your JavaScript application.

Installation

Just include the js before any other javascript. You can include it later, but you will not know all the stuff that your Javascript application is adding to the global namespace

API

GlobalAnalyzer.snapshot(); Display what happened between the last call of the function

GlobalAnalyzer.saveState( 'name' ); Save the current state of the global namespace, in order to compare it at a later moment

GlobalAnalyzer.compareState( 'name' ); Compare the current state with a state previously saved
